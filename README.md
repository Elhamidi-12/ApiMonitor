# ApiMonitor

## Setup

### Prerequisites
- .NET SDK (recommended: .NET 8)

### Restore
```bash
dotnet restore
```

### Build
```bash
dotnet build
```

## Run

### Run the web app
```bash
dotnet run --project src/ApiMonitor.Web/ApiMonitor.Web.csproj
```

### Run with hot reload (optional)
```bash
dotnet watch --project src/ApiMonitor.Web/ApiMonitor.Web.csproj
```
